Panel de control de Barbería

Mantén presionado el **Logo** por 6 segundos para acceder a las opciones de control. Esto evita que personas no autorizadas modifiquen la configuración.

Buscador de Reservas

Esta herramienta te permite encontrar cualquier reserva de forma rápida. Puedes buscar por:
**Año**, **Mes**, **Día**, **Hora** o **Nombre** del cliente.

Opciones de Control

Opciones de Control

  • Abrir Barbería

    Activa la agenda para que los clientes puedan reservar y quita el letrero de "CERRADO".

  • Cerrar Barbería

    Oculta el calendario y muestra un letrero de "CERRADO" para que nadie pueda reservar.

  • Gestionar Horarios por Día

    Permite seleccionar un día específico para cambiar sus horarios, ideal para días festivos o especiales.

  • Borrar todas las reservas

    Una opción para eliminar todas las reservas de la página en caso de una emergencia.

  • Marcar Días Cerrados 1x1

    Haz clic en un día del calendario para deshabilitarlo. Se tornará de color rojo para que nadie pueda reservar ese día.

  • CERRAR/ABRIR Días Recurrentes

    Controla los días de la semana que se cierran o abren de forma regular, como los domingos.

  • Borrar Reservas Antiguas del Mes

    Elimina las reservas desde el primer día del mes hasta el día actual, limpiando el registro rápidamente.

  • Eliminar solo 1 reserva

    Borra una única reserva a tu elección sin afectar a las demás.

Limpieza Automática

Todas las reservas con más de 1 año de antigüedad se eliminarán automáticamente para mantener el rendimiento del sistema y evitar que la base de datos se llene.
